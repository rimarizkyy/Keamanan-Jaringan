**LATAR BELAKANG**

Di pertemuan kali ini, dibahas mengenai IDS dan IPS. Jadi di pembahasan akan dibahas satu – persatu secara mendalam. Untuk IDS merupakan singkatan dari Intusion Detection System. Sementara IPS adalah singkatan dari Intrusion Prevention System.

**PEMBAHASAN**

**IDS atau Intrusion Detection System**

Definisi

Merupakan sebuah software atau hardware yang berguna untuk mendeteksi aktivitas yang sekiranya mencurigakan dalam sebuah jaringan atau sistem. Cara agar IDS mencari bukti terhadap aktivitas mencurigakan adalah dengan cara melakukan inspeksi atau pemeriksaan terhadap aktivitas data masuk dan data keluar.

Jenis IDS dibagi dua yaitu:

- --Network-based Intrusion Detection System (NIDS)

NIDS adalah penganalisis apakah ada aktivitas mencurigakan di semua lalu lintas sebuah jaringan. NIDS diibaratkan &quot;pintu masuk&quot; sebuah jaringan karena terletak pada tempat server berada.

- --Host-based Intrusion Detection System (HIDS)

HIDS adalah sebuah aktivitas host yang bertugas memantau aktivitas mencurigakan seperti serangan dan penyusupan. HIDS terletak pada server yang penting sama seperti letak firewall.

Beberapa HIDS dan NIDS

- --Snort
- --Cybersafe
- --SecureWorks
- --IntruVert
- --NFR
- --RealSecure dari ISS
- --Cisco Secure Intrusion Detection System dari Cisco Systems
- --eTrust Intrusion Detection dari Computer Associates

**IPS atau Intrusion Prevention System**

Definisi

Merupakan kombinasi antara metode IDS dan teknik firewall yang bagus. Biasanya digunakan untuk mencegah serangan yang masuk ke dalam jaringan dengan cara memeriksa semua paket data. Jika serangan memand ada dan telah terdeteksi maka IPS akan melakukan pemblokiran pada log paket data tersebut.

Network-based Intusion Prevention System (NIPS)

Berbeda dengan NIDS, NIPS merupakan gabungan dari fitur IPS dengan firewall  dan juga kadang disebut dengan GIDS atau Gateway Intrusion Detection System.

PENUTUP

1. Kesimpulan : IPS dan IDS merupakan layanan pada keamanan jaringan yang mendeteksi adanya serangan atau penyusupan pada suatu jaringan tertentu. IDS terbagi dua jenis sememntara IPS hanya ada satu jenis saja.
2. Saran : disarankan untuk mencari referansi lain yang lebih mendalam untuk pembahasan IDS dan IPS ini.
